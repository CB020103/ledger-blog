Alongside the deal concepts, a separate bucket of stuff has been more
about tools and habits — the practical, unglamorous skills that don't
show up in a textbook but genuinely make the work faster and better.

## Excel, leveled up

I already knew the basics, but this internship pushed me into territory
I hadn't touched:

- **INDEX/MATCH** — more flexible than VLOOKUP once you actually need to
  look left, not just right, in a table
- **INDIRECT** — referencing a cell by a text string instead of a fixed
  reference, which sounds abstract until you see it used to build a
  dynamic model that pulls from different tabs based on a dropdown
- Building rent comps and full underwriting models from scratch, not
  just filling in someone else's template

## Learning by watching, then doing

A pattern that's worked well: shadow someone senior on a task first
(an underwriting model, a debt analysis, a BOV comparison), then
immediately try building a similar version myself. The gap between
"I watched this" and "I can actually do this" is bigger than I
expected — watching a model get built makes it look easy in a way that
building one from a blank sheet does not.

## Touring properties actually changes how you read a rent roll

I've toured a handful of multifamily properties so far, and the
difference between reading a rent roll on a screen and actually walking
a property is bigger than I expected. Seeing unit finishes, layout, and
condition in person makes it much clearer *why* two properties charging
similar rents can have very different long-term prospects — one might
be newer with smaller units, another older but larger, and the rent
roll alone doesn't tell you which is the better bet.

## A habit worth stealing

One piece of advice that's stuck with me: when you're talking to a
client about a property and its surrounding market, it's not enough to
talk about general trends — you need to back it up with real,
specific numbers. It's a small thing, but it's the difference between
sounding like you have an opinion and sounding like you've actually done
the work.

## Building comparison tools

Also got into building a buyer-side value (BOV) comparison model in
Excel — structuring a way to line up multiple broker opinions of value
side by side, which meant a good amount of practice with structuring a
workbook so it's actually easy for someone else to follow, not just
functional for me.

## What I want to get better at

Speed, mostly — building models like these still takes me meaningfully
longer than the people who've been doing it for years. That gap closes
with reps, so that's really just a function of doing more of it.
